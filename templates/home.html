This is a sample application to demonstrate the following techniques:

  • Using Flask, a Python-based webapp framework
  • Using Heroku for hosting that app
  • Using Github OAuth for user logins
  • Restricting user logins to members of the github org: {{ github_org }}
  • Using MongoDB, a NoSQL database hosted on mlab.com, for database storage.

The source code is available on github at https://github.com/ucsd-cse-spis-2016/spis16-webapps-oauth-and-pymongo-v1

This is provided for participants in UCSD CSE SPIS 2016.
